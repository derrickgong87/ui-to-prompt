# Atlas Notes design system

## Visual north star

Create a warm editorial evidence workspace that feels precise, inspectable, and actively composed rather than magically generated.

## Palette

- Canvas: `#F4F1E8`
- Surface: `#FFFEFA`
- Ink: `#191A17`
- Muted: `#686A62`
- Line: `#D7D2C5`
- Action: `#3157F5`
- Evidence signal: `#DDF45A`
- Warning: `#F05A3C`
- Success: `#1E7A55`

## Typography

Use a compact grotesk display face, a highly legible UI sans, and a mono face for provenance. Keep exact source fonts replaceable; prefer open-source substitutes with similar width and x-height.

## Structure

Marketing surfaces use an asymmetric editorial grid. Workbench surfaces use strict navigation, evidence, preview, and export structure. Components rely on numbered sections, fine rules, evidence classes, and explicit actions.

## Avoid

Do not use purple gradients, glassmorphism, generic centered SaaS card rows, AI robots, glowing orbs, emoji icons, copied source assets, or unverified pixel-perfect claims.
