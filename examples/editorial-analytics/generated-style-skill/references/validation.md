# Validation

Before completion:

- Check 1440 × 960, 1024 × 768, and 390 × 844 layouts.
- Confirm there is no horizontal page overflow.
- Confirm every control is keyboard reachable and has a visible focus state.
- Confirm body contrast, touch targets, semantic labels, and reduced-motion behavior.
- Confirm evidence statuses remain visible and unknowns remain replaceable.
- Confirm no source brand, text, image, logo, or protected asset was copied.
- Confirm the page is real, editable UI rather than a screenshot used as a background.
