# Atlas Notes editorial style

Build interfaces that feel like an actively edited research notebook: warm, precise, calm, and traceable.

## Core invariants

- Use warm paper as the default canvas and reserve white for focused work surfaces.
- Use cobalt for action and selection; reserve signal lime for evidence marks.
- Establish hierarchy with type scale, rules, and asymmetric composition instead of glass effects and stacked shadows.
- Expose whether important conclusions are observed, computed, inferred, translated, user-provided, or unknown.
- Use original content and imagery. Do not copy source branding or protected assets.

Read `references/design-system.md` for the design system, `references/systematic-prompt.md` for the complete implementation prompt, `references/evidence.json` for provenance, and `references/validation.md` before calling a result complete.

When a rule is marked unknown, use a conservative, replaceable fallback and label the assumption. Do not convert uncertainty into a framework default without saying so.
